# node-fs
### fs.js
Usage of Readable Stream for reading files
```
$ node fs
```
### pipe.js
Custom Pipe realisation and Node Pipe usage (sendFile() - can work with live server )
```
$ node pipe
```
### read.js
Simple fs methods for read, cread, remove, rename file
```
$ node read
```
### server.js
Safe sending file example
```
$ node server
```
